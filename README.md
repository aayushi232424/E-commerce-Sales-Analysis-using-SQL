# E-commerce Sales Analysis using SQL

## Project Overview

This project analyzes an e-commerce sales dataset using PostgreSQL. The objective is to extract meaningful business insights using SQL queries.

## Tools Used

- PostgreSQL (pgAdmin)
- SQL
- GitHub

## Dataset

- E-commerce Sales Dataset (1000 Records)

## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, AVG, COUNT, MAX)
- CASE Statement
- HAVING
- Subqueries
- Common Table Expressions (CTE)
- Window Functions

## Business Insights

- Total Revenue
- Total Orders
- Average Order Value
- Revenue by Category
- Top Selling Products
- Payment Method Analysis
- Monthly Sales Analysis
- Highest Spending Customers
- Customer Review Analysis
